The Curious Outsider is a public window into a living knowledge base. It publishes lessons drawn directly from ongoing research and documents the investment practice it supports.

Content begins as plain Markdown to ensure portability, durability, and simple version history. Notes are developed in Obsidian—a local vault under full control—using wiki-style links, backlinks, tags, and a graph view for structure and discovery. Lightweight organization (folders, tags, Maps of Content), daily notes, templates, and selective plugins keep capture fast and navigation clear.

Only selected notes are prepared for publication while preserving internal links and hierarchy, so research threads remain intact. A single Markdown source powers both the private notebook and the public site, which presents curated reference pages, research hubs, and evolving essays.

Published files move to a GitHub repository for versioning and automation. Jekyll builds the site using the Just-the-Docs theme, providing sidebar navigation, built-in search, and readable typography that can adjust as the structure evolves.
